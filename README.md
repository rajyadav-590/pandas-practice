# Pandas Practice

This repository contains my hands-on practice with the **Pandas** library using Google Colab. It covers the fundamental operations used for data analysis and preprocessing.

## Topics Covered

- Creating Pandas Series
- Creating DataFrames
- Reading CSV and JSON files
- Exploring DataFrames
- Handling Missing Values
- Data Cleaning
- Replacing Values
- Mean, Median and Mode
- Date Formatting
- Removing Duplicate Rows
- Correlation Analysis
- Data Visualization using Matplotlib

## Technologies Used

- Python 3
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Dataset

Dataset Source:
https://www.kaggle.com/datasets/themrityunjaypathak/pandas-practice-dataset

The dataset is placed inside the `data/` folder.

## How to Run

1. Clone the repository.

```
git clone https://github.com/yourusername/Pandas-Practice.git
```

2. Install the required libraries.

```
pip install -r requirements.txt
```

3. Open `pandas_practice.ipynb` using Jupyter Notebook or Google Colab.
